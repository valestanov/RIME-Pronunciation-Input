基于国际音标的泰语输入法 - Readme
作者：刘昊文

使用要求：
    本泰语输入法基于Rime输入法（又名中州韵输入法、小狼毫输入法等。网址： https://rime.im/ ）开发，支持Windows/MacOS/Linux平台。
    使用前，请先安装Rime输入法，并阅读该输入法的基础使用说明。
    随后请将本压缩包内的两个yaml文件复制到安装Rime输入法时，您所指定的用户文件夹下。
    用户文件夹可以通过以下方式找到：切换到Rime输入法，在右下角的图标”中“上点击右键，选取用户文件夹。
    复制到用户文件夹后，请右键“中”，打开 输入法设定 ，并在左侧找到 泰语IPA输入法 一项，点击勾选相应的复选框。
    随后，您可以通过点击F4，切换并启用本输入法。
    如果您不是Windows用户，请阅读Rime输入法的使用说明，并按照说明书给出的步骤完成上述操作。

使用前请务必阅读以下说明！
输入说明：
    零声母对应的/ʔ/一律不用输入。
    两个带有/tɕ/的声调均改写成c，即c和ch。
    汉语拼音y对应的辅音，应按照国际音标的要求写作j。
    区分长短音。长元音将该元音输入两次即可。
    特殊音标请通过以下方法输入：
    ɛ和ɔ —— shift+e和o（即大写的e和o）
    ɤ——分号
    ɯ——冒号
    ŋ——输入ng会自动转换
    在每个音节末尾输入以下字母代表声调：
    第一声33——y
    第二声21——z
    第三声41——q
    第四声45——x
    第五声13——v

例词：
    你好 สวัสดี = sa²wat²dii¹ ，请输入 sazwazdiiy
    老师 อาจารย์ = aa¹caan¹ ，请输入 aaycaany
    兄弟 พี่น้อง = phii³nɔɔŋ⁴ ，请输入 phiiqnOOngx
    中级 มัธยม = mat⁴jom¹ ，请输入 matxjomy
    大学 มหาวิทยาลัย = ma⁴haa⁵wit⁴tha⁴jaa¹laj¹ ，请输入 maxhaavwitxthaxjaaylajy

输入时省略声调：
    本输入法强制要求声调，不输入声调无法正确查找到对应的泰语单词。
    作者认为，输入时可以省略声调的话，对初学者的泰语学习十分不利。
    如果想省略声调的话，请使用者自行打开thaiipa.schema.yaml文件，并按照其中speller和translator部分的注释操作。
    yaml文件可以通过记事本打开。
    更改完成后，请右键点击右下角的图标”中“，并点击 重新部署 ，以使更改生效。
    想要重新启用强制声调，反向操作即可。

往词库里添加词汇：
    本输入法并非专业的泰语输入法，仅供泰语学习者使用，只能输入词库内包含的约63000个单词（含部分常用专有名词）。
    如果想要向词库中添加词汇，可以用记事本打开thaiipa.dict.yaml文档，在省略号以下的词库部分任意位置回车添加一行，输入相应的泰文，然后按tab键添加一个制表符，再输入对应的国际音标。
    注意：泰文和国际音标之间的符号必须是制表符，也就是按Tab键添加的那个符号。不能通过按多次空格键在两者之间添加空格，否则会导致输入法无法识别，引发输入法故障！
    Tab键的位置在Caps Lock（大小写切换键）的上方。
    由于dict文件较大，打开和更改可能较为缓慢，属正常现象。您也可以使用PilotEdit等文本文件读取软件打开它。
    请注意，添加的国际音标在拼写风格上需要与词库内的其他词保持一致，且不能带有音节之间的空格或横线等分隔符，否则可能导致无法正常输入。
    更改完成后，请右键点击右下角的图标”中“，并点击 重新部署 ，以使更改生效。
